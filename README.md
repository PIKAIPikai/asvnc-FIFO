# async_fifo
Verilog 硬件描述语言实现的**双时钟异步FIFO**模块，采用**格雷码指针+两级寄存器同步**解决跨时钟域亚稳态问题，是FPGA/ASIC数字设计中跨时钟域数据缓存的经典通用IP核。

## 核心特性
1. **双时钟异步架构**：读写端口独立时钟，支持任意频率跨时钟域数据交互
2. **格雷码指针同步**：二进制指针转格雷码，单次仅1bit跳变，杜绝亚稳态
3. **CDC两级同步器**：读写指针跨时钟域两级寄存器打拍，保证数据传输安全
4. **全参数化配置**：FIFO深度(`DEPTH`)、数据位宽(`WIDTH`)灵活配置
5. **智能位宽计算**：内置`clogb2`函数，自动计算地址位宽，无需手动修改
6. **模块化存储**：内置独立双端口RAM模块，解耦设计，复用性强
7. **安全空满判断**：格雷码对比实现空/满标志，防溢出、防空读
8. **异步复位**：读写端口独立低电平异步复位，上电自动清零

## 参数配置
| 参数名   | 功能           | 默认值 |
| -------- | -------------- | ------ |
| DEPTH    | FIFO存储深度   | 64     |
| WIDTH    | 数据位宽       | 16     |

## 端口信号定义
| 端口名   | 方向   | 功能                     |
| -------- | ------ | ------------------------ |
| i_wclk   | 输入   | 写端口独立时钟           |
| i_wrstn  | 输入   | 写端口异步低电平复位     |
| i_wren   | 输入   | 写使能信号（高电平有效） |
| i_wdata  | 输入   | 写数据总线               |
| i_rclk   | 输入   | 读端口独立时钟           |
| i_rrstn  | 输入   | 读端口异步低电平复位     |
| i_rden   | 输入   | 读使能信号（高电平有效） |
| o_rdata  | 输出   | 读数据总线               |
| o_full   | 输出   | FIFO满标志（高电平有效） |
| o_empty  | 输出   | FIFO空标志（高电平有效） |

## 代码架构
本项目采用**两层模块化设计**，逻辑解耦、易维护：
1. **async_fifo**：顶层模块，实现读写指针控制、格雷码转换、CDC同步、空满标志判断
2. **ram**：底层存储模块，双时钟简单双端口RAM，负责数据读写存储

## 设计亮点
✅ **亚稳态防护**：格雷码+两级同步器，工业标准跨时钟域解决方案
✅ **纯RTL实现**：无厂商IP依赖，兼容Xilinx/Intel/Lattice全系列FPGA
✅ **时序优化**：时序逻辑转换格雷码，降低组合逻辑延迟，支持高频时钟
✅ **代码规范**：注释完整、结构清晰，适配教学、毕业设计、工程开发
✅ **安全机制**：读写使能联动空满标志，避免数据异常

## 适用场景
- 跨时钟域数据交互与隔离
- FPGA多时钟域数据流缓存
- 异步数据缓冲、速率匹配
- 数字IC/ASIC片上跨时钟域存储
- FPGA异步FIFO教学与实验
